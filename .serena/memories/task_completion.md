# Task Completion

To consider a task completed, perform the following validation steps:
1. Run Type Check / Lint:
   `npm run lint`
   Ensure no TypeScript errors or syntax issues.
2. Build the project:
   `npm run build`
   Ensure the production build is successful and output bundling works.
3. Test locally if applicable (no test runner exists yet, so manual testing/Vite preview if needed).
